===============================
Kimchi - a personal dictionary
===============================

1. Description
====================================================
Kimchi allows users to create an unlimited number of 
dictionaries, each of them with a variable number
of customizable columns. The data previously entered
can be retrieved using the search field for each 
column


2. Dependencies
=====================================================
 - Python (developed & tested with 2.6.x) 
 - PyQt (developed & tested with 4.7.x)
   - Qt (required by PyQt)
 - sqlite3  

Developed and (only) tested on KUbuntu 9.10 & 10.04
